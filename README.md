# Trainr

## Description

<p>Trainr is a health and fitness web app that provides all the benefits of existing diet and workout trackers, all for free.<p>

## Diet Tracker

<p>The diet tracker includes a barcode scanner, allowing you to easily identify what you are putting in your body with only your camera.<p>

## Workout Tracker

<p>The workout tracker allows you to make note of everything you've done in the gym, as well as allow you to set reminders of when to go work out<p>
